# Vagrant

Base Vagrant configuration to quickly set a dev environnement.
The "boxes" directory is empty, you can put your own boxes in it and call them in the Vagrantfile.

## "dev" machine
The machine named "dev" contains the following :
- Apache server
- PHP FPM
- Composer
- Nodejs
- Bower
- Gulp
- MySQL
